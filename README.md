🌦️ Weather App

A simple and responsive Weather Application built using HTML, CSS, and JavaScript to learn how to work with APIs and fetch real-time weather data.

📌 Project Overview

This project was created as a learning exercise to understand:

API integration using JavaScript
Fetching data from external services
Handling asynchronous requests with fetch()
DOM manipulation
Responsive UI design with HTML & CSS
Error handling for invalid user input

The application allows users to search for a city and view current weather information instantly.

Search weather by city name

Show weather conditions

Weather icons based on conditions
Responsive design for mobile and desktop

🔗 API Integration

The application fetches weather data from a weather service API using JavaScript's Fetch API.

Example workflow:

User enters a city name.
JavaScript sends a request to the weather API.
API returns weather data in JSON format.
The UI updates dynamically with the received information.
